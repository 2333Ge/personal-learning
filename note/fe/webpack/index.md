## 概念

JS应用程序的静态模块打包工具。将项目中所需的每一个模块组合成一个或多个bundles。

- 入口
- 输出
- loader： loader让webpack能够处理其他类型的文件，并转化为有效模块
- 插件：执行范围更广的任务，如打包优化、资源管理
- mode: 通过选择development, production 或 none 启动相应环境下的优化
