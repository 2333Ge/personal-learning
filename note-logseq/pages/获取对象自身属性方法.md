## 获取对象自身属性的方法对比

| 方法 | 包含不可枚举？ | 包含 Symbol？ | 包含原型链？ |
| ---- | ---- | ---- |
| `Object.keys(obj)` | ❌ 否 | ❌ 否 | ❌ 否 |
| `Object.getOwnPropertyNames(obj)` | ✅ 是 | ❌ 否 | ❌ 否 |
| `Object.getOwnPropertySymbols(obj)` | ✅ 是 | ✅ 是 | ❌ 否 |
| `Reflect.ownKeys(obj)` | ✅ 是 | ✅ 是 | ❌ 否 |
| `for...in` | ✅ 是 | ❌ 否 | ✅ 是（可枚举） |
