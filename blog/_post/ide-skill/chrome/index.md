## 快捷键

|   快捷键    |       描述       |
| :---------: | :--------------: |
| cmd+shift+f |     切换全屏     |
| cmd+shift+b | 显示、隐藏书签栏 |


## 插件

- Timer: 倒计时提醒，帮助定制时间计划
  浏览器输入 `timer + Tab + 倒计时分钟数`,固定插件后才可显示剩余时间
  ![timer](./chrome-plugin-timer.png)
- Cookie-Editor cookie导出 导入
- 油猴
- 繪畫工具和屏幕截圖：取色器，画草稿
- Quick QRCode：二维码生成
